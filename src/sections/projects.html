To do List Web

Modern Todo List Web App is a responsive task management application built with HTML, Tailwind CSS, and Vanilla JavaScript, featuring a modern glassmorphism interface and Dark/Light Mode support. The app allows users to create tasks with precise date and time inputs, automatically sorts tasks by nearest schedule, supports status control (Done/Pending), and provides both single and bulk delete actions. It also includes real-time search and dynamic filtering (All, Pending, Done) with a responsive table view, making task tracking more organized and efficient.

GastroCare

Edusign

Edusign (GamSign) is an Android-based learning application designed to help the general public learn SIBI (Indonesian Sign Language) through an interactive and accessible approach. The app integrates computer vision using a customized YOLOv11 model to recognize hand signs, combined with gamification elements such as levels, challenges, and interactive learning flows to improve user engagement. Built to support inclusive education, the application is optimized for smartphones with at least 4 GB RAM, 64 GB storage, and Android 13, aiming to encourage broader community participation in creating a more inclusive communication environment.
